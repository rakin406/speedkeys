## Description

WPM (Words Per Minute) counter and visualizer inspired by various typing
web applications.

Framework used: [Nest](https://github.com/nestjs/nest)

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
